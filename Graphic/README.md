### 图不连通怎么办
**连通**：如果从v到W存在一条（无向）**路径**，则称v和w是连通的

**路径**：v到W的路径是一系列顶点{v,v1,v2,...,vn,w}的集合，其中任意一堆相邻的顶点
都有图中的边。**路径的长度**
